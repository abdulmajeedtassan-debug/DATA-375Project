Music Data Analysis Vignette

Introduction

This vignette shows how to use the musicDataAnalysis package. The package helps clean, summarize, and visualize the Top Spotify Songs 2023 dataset. The goal is to understand patterns in music popularity using streams and audio features such as energy, danceability, and bpm.

Load the package and sample data

library(musicDataAnalysis)

spotify_sample <- load_spotify_sample()

Summary statistics

summarize_music(spotify_sample)
#>   avg_streams correlate_energy correlate_dance correlate_bpm
#> 1   514137425      -0.02605149      -0.1054569  -0.002437908

This output gives the average number of streams and the correlations between streams and the audio features.

summarize_features(spotify_sample)
#>        feature      mean       sd min median max
#> 1       energy  64.27416 16.55852   9     66  97
#> 2 danceability  66.98424 14.63128  23     69  96
#> 3          bpm 122.55357 28.06960  65    121 206

This output gives basic statistics for energy, danceability, and bpm.

Visualizations

plots <- plot_music(spotify_sample)
plots$scatter_energy_streams

The scatterplot helps show the relationship between energy and streams.

plot_feature_distribution(spotify_sample, "bpm")

This plot shows the distribution of bpm across songs.

Conclusion

This package gives users a simple way to clean Spotify data, calculate basic summaries, and create useful plots. These tools make it easier to explore what characteristics appear in popular songs.
